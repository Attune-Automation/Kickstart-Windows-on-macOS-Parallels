The following steps will guide you through the manual process.

Step 1 - KS Clean Build Files (link)

Deletes the build files and kickstart ISO.

Step 2 - KS Check macOS for ISO Building (link)

Checks if the genisoimage is aliased with xorriso.

Step 3 - Deploy Windows Desktop 10 ISO (link)

Deploys Windows Desktoop ISO to ${HOME}/Parallels/os_auto_install/build-{kickstartedNode.fqn}.

Step 4 - KS Rename ISO (link)

Renames the ISO at directory ${HOME}/Parallels/os_auto_install/build-{kickstartedNode.fqn to kickstart.iso.

Step 5 - KS Deploy Win10 Unattended Config for Parallels (link)

Deploys the Windows Desktop 10 unattended config for Parallels Desktop.

Step 6 - KS Deploy Parallels Drivers (link)

Deploys the Parallels Desktop drivers to ${HOME}/Parallels/os_auto_install/build-{kickstartedNode.fqn}/Drivers.

This will be seen at the second CD ROM at E:\Drivers by Autounattend.xml.

Step 7 - LIN Deploy Custom Setup PS1 Script (link)

Deploys the custom run once PowerShell scripts.

Step 8 - KS Create Drivers and Answer File ISO for Parallels (link)

Creates an ISO called Drivers.iso containing:

  1. The Drivers folder.
  2. Autounattend.xml
  3. custom_setup.ps1

This works for Parallels Desktop kickstarts with: 1. Windows Desktop 10 2. Windows Server 2019 3. Windows Server 2022

Step 9 - KS Parallels Recreate Windows Virtual Machine (link)

Recreates a Parallels Desktop virtual machine using:

  1. 1 x unaltered Windows ISO.
  2. 1 x drivers ISO.

Step 10 - Remove CD ROM Drives and Restart (link)

Removes both CD ROMs.

Step 11 - KS Clean Build Files (link)

Deletes the build files and kickstart ISO.

Step 12 - Install Parallels Tools (link)

Install Parallels Tools. Use this after Windows installation has completed.
